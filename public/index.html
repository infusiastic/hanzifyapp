Hanzify!

by Demian ”Infusiastic“ Terentev

How to use

This web service transliterates any given name into Chinese according to official standard. So far only Russian names are supported. The name can be supplied using the form on this page or directly in the URL (the result will be shown on separate page), e.g:

Мухосранск

Как пользоваться

Данный веб-сервис транслитерирует любое заданное название или имя по-китайски в соответствии с официальным стандартом. В настоящее время поддерживаются только русские названия и имена. Название или имя можно задать как с помощью формы на данной странице, так и напрямую в адресе (результат покажется на отдельной странице), например:

Мухосранск
